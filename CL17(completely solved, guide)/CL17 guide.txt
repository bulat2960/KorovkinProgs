Задание:
CL17. Целочисленный массив произвольного размера заполнить
нулями и единицами, расположив их в шахматном порядке.

Решение:
1. Стандартно объявляем матрицу, проверяя корректность введенных пользователем размерностей.
2. Заполняем матрицу по принципу цветов шахматной доски. Принцип таков: если сумма индексов матрица кратна двум, "красим" ячейку матрицы в один цвет, 
иначе в другой. Так как цветов требуется два, их роль выполняет булевская матрица.
3. Выводим матрицу на экран, получаем требуемое.

NOTE: для тех, кому непонятна эта строчка (i + j) % 2 == 0 ? mtrx[i][j] = true : mtrx[i][j] = false;
Это тернарный оператор, который является аналогом if else. Работает так: Выражение ? выполнить если выражение истина : выполнить если выражение ложь
Для тех, кому непонятно, можете переписать на понятный вид:
if ((i + j) % 2 == 0)
{
	mtrx[i][j] = true;
}
else
{
	mtrx[i][j] = false;
}
