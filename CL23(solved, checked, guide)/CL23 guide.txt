Задание:
CL23. Клеточное поле размером (m x n) есть результат игры в
крестики-нолики. Проверить, не закончена ли игра выйгрышем
«крестиков»? Считается, что крестики выиграли, если на поле найдется по
горизонтали, вертикали или диагонали цепочка, состоящая подряд из
k крестиков.

Решение:
1. Вводим с клавиатуры размеры матрицы и цепочку необходимой длины.
2. Генерируем матрицу посредством вызова функции generateMatrix.
3. Отображаем матрицу на экране с помощью showMatrix.
4. Вызываем по очереди функции проверки по строке, столбцу, главной и побочной диагонали. Если одна из функций возвращает истину,
завершаем работу программы.
5. Принцип работы функций проверки: бежим по всем элементам матрицы, кроме тех, которые и так будут проверены
Пример: матрица 5х5, необходимо найти в строке цепочку длиной 3, следовательно, достаточно бежать в строке по элементам от 0 до 2, так как
при элементе [0, 0] будет проверена цепочка [0, 0]-[0, 1]-[0, 2], при элементе [0, 1] будет проверена цепочка [0, 1]-[0, 2]-[0, 3], и при элементе
[0, 2] будет проверена цепочка [0, 2]-[0, 3]-[0, 4], то есть мы дойдем до конца матрице по строке. По схожей аналогии работает проверка по столбцу, 
главной и побочной диагоналям.
