Задание:
CL20. Написать наблонную функцию для вычисления определителя
квадратной матрицы произвольной размерности на основе метода Гаусса.

Решение:
1. Стандартно объявляем матрицу, делаем проверку на корректность размеров.
2. Инициализируем матрицу вызовом функции generateMatrix.
3. Выводим матрицу на экран с помощью showMatrix.
4. Вызываем функцию determinantMatrix, которая считает определитель матрицы по методу Гаусса. Если в ходе работы этой функции окажется, что один из 
элементов главной диагонали равен нулю - сгенерировать исключение с ошибкой. Если входная матрица имеет размер 2х2, считать ее стандартным способом.
Принцип работы функции вычисления детерминанта:
Положим, задана матрица 3х3
х1 х2 х3
х4 х5 х6
х7 х8 х9
Циклы работают так, что сначала обнуляются элементы первого столбца:
x1 x2 x3
0  y1 y2, где y1 = x5 - x5*(x4/x1), y2 = x6 - x6*(x4/x1), y3 = x8 - x8*(x7/x1), y4 = x9 - x9*(x7/x1)
0  y3 y4
Далее второго:
x1 x2 x3
0  y1 y2, где z1 = y4 - y4*(y3/y1)
0  0  z1

5. Ответом является произведение элементов на главной диагонали: x1*y1*z1.
